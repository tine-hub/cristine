Computer Terms

CPU
The Central Proccesing unit, responsible for executing instruction in a computer.
RAM
Random Access Memory, used by computers to store working data temporarily.
SSD
Solid State Drive, a faster storage device that used flash Memory to store data.
GPU
Graphics Processing Unit, specialized for rendering images and video
Motherboard
The main circuit board of a computer, connecting all components.
Router
A networking device that forwards data packets between computer networks, identifying it for communication
IP Adress
A unique string numbers assigned to every device connected to a network, identifying it for communications.
